# Wolf_Design_sales_dashboard
Wolf Design Sales Intelligence Dashboard  An interactive Excel Business Intelligence  dashboard analyzing 1,000+ sales transactions  for Wolf Design clothing brand across 4 Indian  cities (Mumbai, Delhi, Hyderabad, Kolkata).
# 📊 Wolf Design Sales Dashboard

## Overview
Interactive Excel BI Dashboard analyzing 
Wolf Design's clothing brand sales performance 
across 4 Indian cities in 2024.

## Key Metrics
- 💰 Total Revenue: ₹2,44,31,017
- 📦 Units Sold: 8,334
- 💹 Total Profit: ₹2,08,98,207  
- 🏷️ Average Discount: 15.44%

## Features
- Interactive slicers (City, Quarter, Year, Rep)
- Revenue & Profit by City
- Category performance analysis
- Monthly revenue trends
- Sales rep performance tracker

## Tools Used
Microsoft Excel — Pivot Tables, Charts,
Slicers, Conditional Formatting, SUMIF,
COUNTIF, AVERAGEIF formulas

## Dataset
1,000 sales transactions
4 cities: Mumbai, Delhi, Hyderabad, Kolkata
6 categories: Jackets, Ethnic Wear, Dresses,
Trousers, Accessories, Shirts
12 Sales Representatives
